# Training Log — Влад, сезон 2026

Операционный репозиторий: планы недель, логи тренировок (зал + бег), сбор метрик из intervals.icu + Garmin, review-цикл.
Работает через две slash-команды в Claude Code: [`/training:sync`](.claude/commands/training-sync.md) и [`/training:weekly`](.claude/commands/training-weekly.md).

## Приоритеты сезона

1. **Бег** — полумарафон конец мая 2026 (1:34–1:40), 10 км < 40:00 и 5 км < 19:00 осенью.
2. **Зал** — первый цикл, реабилитация колена (гонартроз 3 ст) + силовая база.
3. **Вес** — 90–92 кг стабильно (семаглутид микродозинг).

Подробно: [goals.md](goals.md) · программа зала: [gym-program.md](gym-program.md) · протокол веса: [weight-protocol.md](weight-protocol.md) · профиль: [me.md](me.md).

## Навигация

- [weeks/](weeks/) — недельные файлы (план + статус + review)

## Как работает цикл

1. **`/training:sync`** (раз в 1–3 дня) — Claude тянет данные intervals.icu + Garmin с последней обработанной даты до сегодня, обновляет активный недельный файл, добавляет сессии в `training/gym.md` и `training/running.md`, шлёт короткий дайджест в Telegram.
2. **Ты в файле** — дописываешь заметки, веса, ощущения, вопросы в колонке «Заметка» или в «Open questions».
3. **`/training:weekly`** (Вс вечер) — Claude анализирует прошедшую неделю, заполняет секцию «Review», создаёт черновик следующей недели из шаблона с учётом твоих целей и формы. Ты проверяешь → подтверждаешь → коммит.
4. Прошлая неделя уезжает в `weeks/archive/`.
